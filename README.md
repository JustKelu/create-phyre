# Create Phyre

Official CLI to create new Phyre projects.

## Usage
```bash
# Create a new project interactively
npx create-phyre@latest

# Create with a specific name
npx create-phyre@latest my-app
```

## What's included?

- Interactive project setup
- TypeScript support
- Package structure option
- Automatic dependency installation

## Templates

- `template` - Basic JavaScript setup
- `ts-template` - TypeScript setup
- `package-template` - Multi-package JavaScript
- `package-ts-template` - Multi-package TypeScript

## Add-ons

- `Docker` - Dockerfile and docker-compose setup
- `Database` - Database integration in your docker-compose

## Requirements

- Node.js >= 18.0.0
